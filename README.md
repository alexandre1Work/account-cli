# Banco 🏦  

Um projeto de um banco no terminal que permite operações bancárias, utilizando `inquirer` para interação com o usuário e `chalk` para estilização no terminal.  

## 🛠️ Funcionalidades  
- 📄 Criar conta  
- 💰 Depositar  
- 💸 Sacar  
- 📊 Consultar saldo  

## 🚀 Tecnologias  
- Node.js  
- Inquirer  
- Chalk
